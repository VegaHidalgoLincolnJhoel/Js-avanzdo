# Trabajos y Proyectos

Carpeta para subir los trabajos y proyectos más extensos del curso.

## Convención de nombres

Cada trabajo debe ir en su propia subcarpeta:

```
trabajos/
  trabajo1-nombre-del-proyecto/
    index.js
    README.md   ← descripción del trabajo
```

## Instrucciones generales

1. Crea una subcarpeta con el nombre del trabajo.
2. Incluye un `README.md` dentro de la subcarpeta que describa:
   - Objetivo del trabajo
   - Cómo ejecutar el código
   - Decisiones de diseño relevantes
3. Asegúrate de que el código esté bien comentado y funcione correctamente.
