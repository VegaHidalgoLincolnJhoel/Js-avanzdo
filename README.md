# JavaScript Avanzado

Repositorio del curso de JavaScript Avanzado para subir tareas y trabajos.

## Estructura del repositorio

```
📁 01-scope-closures/     → Ámbito y clausuras
📁 02-prototypes/         → Prototipos y herencia
📁 03-async-callbacks/    → Asincronía y callbacks
📁 04-promises/           → Promesas
📁 05-async-await/        → Async / Await
📁 06-es6-features/       → Características modernas de JS (ES6+)
📁 tareas/                → Tareas individuales
📁 trabajos/              → Trabajos y proyectos
```

## Temas del curso

1. **Scope y Closures** – variables, hoisting, clausuras
2. **Prototipos** – cadena de prototipos, herencia prototípica
3. **Asincronía y Callbacks** – event loop, callbacks, manejo de errores
4. **Promesas** – creación, encadenamiento, Promise.all / Promise.race
5. **Async / Await** – sintaxis moderna para código asíncrono
6. **ES6+** – arrow functions, destructuring, módulos, spread/rest, clases

## Cómo subir una tarea

1. Coloca tu archivo dentro de la carpeta del tema correspondiente o en `tareas/`.
2. Nombra el archivo de forma descriptiva, por ejemplo: `tarea1-closures.js`.
3. Haz commit y push de tus cambios.
