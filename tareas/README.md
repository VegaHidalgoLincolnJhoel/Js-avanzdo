# Tareas

Carpeta para subir las tareas individuales del curso.

## Convención de nombres

Usa el siguiente formato al nombrar tus archivos:

```
tarea<número>-<tema>.js
```

### Ejemplos

- `tarea1-closures.js`
- `tarea2-prototypes.js`
- `tarea3-promises.js`

## Instrucciones generales

1. Cada tarea debe estar en su propio archivo `.js`.
2. Incluye comentarios que expliquen tu solución.
3. Verifica que tu código funcione antes de hacer el commit.
